# 毛球 — 宠物社区 + 本地服务平台

毛球是一个面向宠物主人的移动应用，核心功能包括宠物日常分享、附近宠物服务、AI 健康问答和用品团购。目标是让养宠更简单、更有趣、更省心。

> **AI 导航**：收到问题时请优先读 `WORKSPACE_MAP.md` 定位相关文档路径，再精准读取目标文档，避免全量扫描。

## 技术栈

- 前端：React Native (App) + React (管理后台)
- 后端：Python (FastAPI) + Go (高并发服务)
- 数据库：PostgreSQL (业务) + ClickHouse (分析) + Redis (缓存)
- AI：OpenAI API (问答) + 自训练推荐模型 (召回+排序)
- 部署：阿里云 K8s + OSS (图片/视频存储)
- 监控：Grafana + Prometheus

| 埋点方案 | `docs/data/推荐_埋点方案.md` | 曝光/点击/互动/转化全链路埋点 |
| 算法沟通纪要 | `docs/prd/推荐_算法沟通纪要.md` | 召回策略、冷启动方案、模型选型讨论 |

### 核心架构速览

- **定位**：首页从时间线改为个性化推荐 feed，作为流量分发中心
- **推荐链路**：请求 → 画像查询 + 内容召回 → 过滤 → 排序 → Top N → 卡片组装 → 展示
- **冷启动**：新用户用宠物档案（品种+年龄）做粗排，3 天后切换协同过滤
- **AI 问答**：宠物健康知识库（RAG），基于用户宠物档案提供个性化回答

## 写入规范（AI 必读）

### 命名规范

- 文档：`项目_描述.md`，中文描述，简短直白
- 带时效的文档加日期：`推荐周报_20260401.md`
- 脚本：英文 snake_case，如 `export_recommend_data.py`

### 目录归属

| 产出物类型 | 放到哪里 |
|-----------|---------|
| PRD / 设计规范 | `docs/prd/` |

**根目录原则**：根目录只放系统文件（CLAUDE.md / WORKSPACE_MAP.md 等）。其他文件按类型放到对应目录。
